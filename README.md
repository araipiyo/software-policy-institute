# Software Policy Institute

Software Policy Instituteは、ソフトウェア領域における技術と政策の関係について、特定の政治的立場や業界利害から距離を置きつつ、技術的観点からの解説・分析を行う機関である。

本サイトの内容は、特定の政策提言や法制度の採否を直接の目的とするものではなく、技術的観点からの論点整理を目的とする。

# 代表者略歴

新井俊一は、1978年東京生まれ、大阪在住のソフトウェア起業家であり、**独立行政法人情報処理推進機構**(IPA)からスーパークリエイターとして認定された技術者でもある。CTO（最高技術責任者）としての実務経験を背景に、技術・経営・政策の交差領域に関する論考を発表している。
